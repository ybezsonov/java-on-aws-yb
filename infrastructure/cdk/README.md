# Welcome to your CDK Java project for Java on AWS infrastructure!

It is a [Maven](https://maven.apache.org/) based project, so you can open this project with any Maven compatible Java IDE to build and run tests.

## Useful commands

In `infrastructure/cdk` folder:
 * `mvn clean package`      compile and run tests
 * `cdk synth`              emits the synthesized CloudFormation template

In `infrastructure` folder:
 * `npm run generate-cfn`   generate CloudFormation template `infrastructure/cfn/UnicornStack.yaml`
